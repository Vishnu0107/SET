# SET
Seismic Exploratory Team (SET) is specialized in detecting the quakes in planets using ML and DSP
# Pre Processing
Using Bandpass filter along with mean frequency to filter for noise.
STA\LTA ratio is used to detect sudden noise spikes and binarize it to 0.
Bandpass filter along with STA\LTA filtering removes most noise spikes
